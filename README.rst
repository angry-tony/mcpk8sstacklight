=======================
MCP Virtual Labs Models
=======================

OpenStack Reclass models for Mk-based cloud deployments for QA.


Available deployments
=====================


MCP 0.5 DVR (Openstack Mitaka)
------------------------------

* 1 config node
* 3 control nodes
* 1 compute node


MCP 0.5 OVS (Openstack Mitaka)
------------------------------

* 1 config node
* 3 control nodes
* 1 compute node
* 1 monitor node


MCP 1.0 OpenContrail (Openstack Mitaka)
---------------------------------------

* 1 config node
* 3 control nodes
* 2 compute nodes
* 1 monitor node
* 1 meter node
* 1 log node
